# Research: Brakujące Grafiki Przedmiotów / Pętla Generowania [IMG-01]
Data: 2026-07-23  
Stack: Next.js 14 (App Router), React 18, TypeScript (strict), Google Gemini API (Imagen / Flux / Lite), Tailwind CSS

## Obszar problemu

1. **`src/app/api/equipment/generate-starting/route.ts`**
   - Generator przedmiotów startowych tworzy obiekty `EquipmentItem` i wzbogaca opisy oraz cenę ($) przez model Gemini Lite.
   - **Brak logiki przypisywania grafik:** Endpoint zwraca przedmioty bez przypisanego pola `imageUrl` lub z obrazami statycznymi szablonu, przerzucając zapotrzebowanie na ilustracje na tło sesji.

2. **`_tester/_base/.silnik/src/hooks/useEquipmentThumbnails.ts`**
   - Hook generowania miniatur w tle (`generateThumbnailsInBackground`) filtruje przedmioty bez `imageUrl` lub z domyślnymi SVG.
   - **Główna przyczyna pętli:** Jeśli endpoint `/api/imagen` lub `/api/flux-kontext` zwróci błąd (brak klucza, quota, błąd sieci, HTTP 429/500), funkcja `generateOneThumbnail` wyłapuje go i zwraca `null`. Kod robi `continue`, nie modyfikując stanu przedmiotu (`item.imageUrl` pozostaje `undefined` lub `.svg`).
   - Przy kolejnym re-renderze/restarcie sesji przedmioty te są ponownie zakwalifikowane do generowania w tle, generując niekończącą się pętlę i niepotrzebne koszty sesji (21 nieudanych/powtórzonych zapytań API).

3. **`_tester/_base/.silnik/src/lib/equipment-catalog.ts`**
   - Posiada statyczny słownik i 35 gotowych plików `.webp` w `public/equipment/catalog/` (np. `flashlight-1920s.webp`, `revolver-1940s.webp`).
   - **Luka w zasobach:** Część przedmiotów z generatora startowego (np. *Koperty na dowody*, *Zniszczona odznaka*, *Pistolet sygnalizacyjny*) nie posiada pasującego wpisu w katalogu `.webp` i dostaje fallback `.svg`. Ikony `.svg` są z kolei odrzucane przez `useEquipmentThumbnails.ts` i wymuszają generowanie AI.

4. **`src/components/ui/equipment-detail-dialog.tsx`**
   - Modal podglądu detalu przedmiotu. Pokazuje grafiki przedmiotu/mapy oraz pozwala czytać dokumenty.

## Zależności

- `src/app/api/equipment/generate-starting/route.ts` -> tworzy przedmioty startowe.
- `_tester/_base/.silnik/src/hooks/useEquipmentThumbnails.ts` -> zarządza kolejką miniatur w tle po starcie gry.
- `_tester/_base/.silnik/src/lib/equipment-catalog.ts` -> dostarcza grafiki statyczne z katalogu WebP i SVG.
